## Security Updates

Enhanced input validation across all search functionality to prevent potential injection attacks. Special characters that could pose security risks are now blocked in search queries.

??? note "Internal references"
    ENG-1096

??? abstract "Internal: Admin, Data & Traceability"

    ### Admin Updates

    **User Role in Manage Employees**
    Company Administrators can now see user roles in the Manage Employees view, making it easier to audit team permissions.

    **Invoice Reference Environment Tags**
    Invoice references now include environment identifiers (DEMO, TEST) for non-production environments, making it easier to distinguish test invoices from production.

    ### Data Fixes

    No data fixes required for this release.
